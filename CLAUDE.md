# 工作规范

- **最小上下文原则**：优先用 subagent 完成任务，主 agent 只做调度与总结，不堆积中间结果
- 先阅读代码再改，不要猜测未检查的代码
- 代码注释用中文，变量命名用英文
- 不确定的地方询问我，不要瞎猜
