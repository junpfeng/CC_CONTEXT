## 做什么

## 涉及端

server / client / both（选一个）

## 触发方式

## 预期行为

1. 玩家做 X
2. 系统响应 Y
3. 最终效果 Z

异常/边界情况：
- 条件不满足时：提示 "xxx" / 静默忽略 / 回退到默认行为
- 重复操作时：覆盖上一次 / 拒绝并提示 / 累加
- 目标不存在时：如何处理
