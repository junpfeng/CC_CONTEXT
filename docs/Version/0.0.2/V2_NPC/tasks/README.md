## 并行策略

- **Wave 1**: task-01/02/03/04（服务端基础）与 task-08/09（客户端）完全并行，无交叉依赖
- **Wave 2**: task-05 和 task-06 共享前置 02/03/04，可并行开发（无文件冲突）
- **Wave 3**: task-07 依赖 05/06 完成；task-10 依赖 01 的数据产物
- **Wave 4**: task-11 集成验证，所有任务完成后执行
