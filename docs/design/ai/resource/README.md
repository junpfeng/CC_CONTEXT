# NPC 资源文件索引

NPC 开发涉及的资源文件路径与说明。源码文件清单见 [`../npc/npc-v2-file-inventory.md`](../npc/npc-v2-file-inventory.md)。

> 路径说明：服务端路径相对于 `P1GoServer/`，客户端路径相对于 `freelifeclient/`，配置表路径相对于 `freelifeclient/RawTables/`。

> 本文档只收录源文件，打表/Proto 生成的输出文件不在此列。

| 文档 | 内容 |
|------|------|
| [server-config.md](server-config.md) | 服务端配置文件（TOML、AI 决策 JSON、行为树 JSON、NavMesh） |
| [excel-tables.md](excel-tables.md) | Excel 配置表源文件（打表输入） |
| [client-assets.md](client-assets.md) | 客户端美术资源（Prefab、动画、音频） |
| [protocol.md](protocol.md) | Proto 协议定义文件 |
